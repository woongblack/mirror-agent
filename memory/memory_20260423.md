1. **패턴화** — 과거 대화 → Extractor → Generalizer → 규칙 DB
2. **초안 기획** — Confluence/아이디어 → Planning Agent (우로보로스 문답) → 1차 기획안
3. **기획 구체화** — Historical Agent 비판 + 방어 예측 → 수용/반려 → 반복
4. **Jira 발행** — 검증된 기획 → Jira 티켓
